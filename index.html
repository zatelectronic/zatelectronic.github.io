ZATINO V2 SMD
KiCad Arduino ATMEGA328P SMD Open Source
// Sobre o projeto
O ZATINO V2 SMD é uma placa Arduino StandAlone desenvolvida com componentes SMD (Surface Mount Device), projetada para ser compacta, eficiente e de fácil integração em projetos embarcados customizados. Todo o design foi desenvolvido no KiCad.
// Características
  • Microcontrolador ATMEGA328P em package SMD
  • Design compacto com componentes de montagem superficial
  • Compatível com bootloader Arduino
  • Projetado para aplicações embarcadas customizadas
  • Hardware open source — arquivos disponíveis no GitHub
  • Desenvolvido com KiCad
// Ferramentas utilizadas
  • KiCad — Esquemático e Layout PCB
  • Arduino IDE / PlatformIO — Firmware
  • C++ — Linguagem de programação
